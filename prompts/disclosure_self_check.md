## 生成 .docx

全部通过后执行：

```bash
python3 tools/mermaid_render.py -i "<发明名称_YYYYMMDDHHmmss>.md" -o "<发明名称_YYYYMMDDHHmmss>.md"
```

默认同目录生成同名 `.docx`；可用 `--docx` 指定路径，`--no-docx` 跳过。Word 失败时按 stderr 提示手动：

```bash
python3 tools/md_to_docx.py -i "<发明名称_YYYYMMDDHHmmss>.md" -o "<发明名称_YYYYMMDDHHmmss>.docx"
```
